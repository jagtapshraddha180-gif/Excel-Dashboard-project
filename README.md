# IPL Excel Dashboard 📊

## 📌 Project Overview
This project is an interactive IPL dashboard created in Microsoft Excel to analyze team performance, player statistics, and match insights.

## 🚀 Features
- Dynamic dashboard with slicers and filters
- Team-wise and player-wise analysis
- Interactive charts and pivot tables
- Data-driven insights

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Data Cleaning
